# 🚚 WeCarry (Taşıyıcın)

A scalable logistics platform connecting customers with carriers via web and mobile applications.

🔗 Live: https://www.tasiyicin.com

---

## 🔒 Source Code

This repository is public for showcase purposes.  
The source code is private due to project confidentiality.

---

## 🚀 Features

- Real-time order tracking
- Courier assignment system
- Web & mobile integration
- RESTful API architecture
- Authentication & authorization system
- Scalable backend structure

---

## 🏗️ Architecture

- Layered (N-Tier) Architecture
- Repository Pattern
- Separation of Concerns
- API-based communication between services

---

## 🧱 Tech Stack

### Backend
- ASP.NET MVC
- Entity Framework Core
- SQL Server

### Mobile
- React Native (Expo)
- REST API integration
- Background location tracking

### Frontend
- HTML, CSS, JavaScript
- Bootstrap

## 🧠 Development Approach

- Continuous improvement and optimization
